# Node.js-realtimechart
Node-js for realtime process with socket io

## library used :
- express framework
- Mongoose (Mongodb Wrapper)
- Socket.io

## this will show realtime barchart
- first install dependency
- run the app 
```bash
node app.js
```
- post using postman, other api tools to 
```bash 
localhost:3000/vote
```
request body:
```bash
name : "name you want"
```
